<p>This example demonstrates how to use <b>notistack.</b><br />notistack has an imperative API that makes it easy to display snackbars, without having to handle their open/close state. <br />It also enables you to stack them on top of one another.<br />(The current recommendation is to use the 'notisnack' library. ### Concept
<p>The `cra-template-moaui` already includes the installation of <b>notistack.</b> <br />Refer to the usage guide below, and simply use `enqueueSnackbar` appropriately in the desired locations!</p>

<br />
### How to limit the number of snacks. (`maxSnack` in `SnackbarProvider`)
<p>By default, the number of snacks is unlimited. <br />If you want to limit the number of snacks, you can use the `maxSnack` property of `SnackbarProvider`.</p>
<p>After installing a `cra-template-moaui`, you can set the `maxSnack` property in `src/index.tsx`.</p>
<Source
	code={`root.render(
	<React.Fragment>
		{/** DEFINE SNACKBAR REGION */}
		<SnackbarProvider maxSnack={3}>
			<App />
			{/** FOR PYTHON SCRIPTING (PYTHON CODE LINKING) */}
			{React.createElement('py-script', { src: './Python/Code.py' })}
		</SnackbarProvider>
	</React.Fragment>
);`}
	language='tsx'
	dark
/>
<p>maxSnack={3} -> Set the maximum number of snacks to 3.<br />
if you want to set it to 1, set it to maxSnack={1}.</p>

<br />
### How to add Snackbar (`useSnackbar` `enqueueSnackbar`)
<p>After installing a `cra-template-moaui`, you can see `useSnackbar` and `enqueueSnackbar` in `src/SampleComponents/ButtonSnackbar.tsx`.</p>
<Source
	code={`import { Button, Stack } from '@midasit-dev/moaui-components-v1';
//(1) import useSnackbar from notistack
import { useSnackbar } from 'notistack';

const SnackComponentSuccess = () => {
	// (2) useSnackbar is a hook of notistack
	const { enqueueSnackbar } = useSnackbar();

	return (
		<Button
			onClick={() => {
				// (3) enqueueSnackbar is a function of notistack, use it
				// enqueueSnackbar(message, options)
				// message: string
				// variant: 'default' | 'error' | 'success' | 'warning' | 'info'
				enqueueSnackbar('Success Message!', { variant: 'success' });
			}}
		>
			Snackbar Success
		</Button>
	)
}`}
	language='tsx'
	dark
/>
<p>(1) Import `useSnackbar` from `notistack`.</p>
<p>(2) define useSnackbar is a hook of notistack.</p>
<p>(3) enqueueSnackbar is a function of notistack, use it</p>
<p>Enable enqueueSnackbar Options: 'default' | 'error' | 'success' | 'warning' | 'info';</p>
